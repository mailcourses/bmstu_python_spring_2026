### 1. Функция анализа температурных данных

Функция должна принимать на вход список чисел, проводить анализ и возвращать словарь с результатами анализа. Подсчитаны должны быть средняя температура, максимальная и минимальная температуры и списки дней с температурой выше средней (сами значения и индексы в исходном списке).
Средняя, минимальная и максимальная температуры должны быть округлены до целого значения.
```
temperatures = [12.5, 15, 10.1, 14.9, 16.234, 18, 13.0]
res = analyze_temperature(temperatures)

expected = {
    "avg": 14,
    "min": 10,
    "max": 18,
    "above_avg": [15, 14.9, 16.234, 18],
    "above_avg_idx": [1, 3, 4, 5],
}

assert res == expected
```


### 2. Слияние отсортированных списков без дубликатов

Реализуйте функцию merge, которая принимает на вход две отсортированные по возрастанию последовательности чисел и возвращает отсортированный список, состоящий из всех элементов обеих последовательностей без дубликатов.
В комментарии фукнции нужно указать оценку времени выполнения данной функции.

```
lst = [1, 1, 2, 5, 7]
tp = (1, 1, 2, 3, 4, 7)
res = merge(lst, tp)
assert res == [1, 2, 7]
```


### 3. Функция поиска контакта

Функция должна принимать на вход список контактов, название и значение поля, по которому будет производиться поиск.
Если найден один или несколько подходящих контактов, их необходимо вернуть в виде списка. Если ничего не найдено, то следует вернуть пустой список.
Набор полей у каждого контакта должен быть одинаковым, при этом сам набор произвольный.
Поиск должен осуществяться по полному совпадению значения поля без учета регистра.

```
contacts = [
    {"name": "Александр Пушкин", "phone": "+7 910 56-78", "email": "pushkin@example.com"},
    {"name": "Стив Возняк", "phone": "+7 123", "email": "iwoz@example.com"},
    {"name": "Брюс Уэйн", "phone": "+7 987 654", "email": "batman@example.com"},
    {"name": "Бэтмен", "phone": "+7 gotham", "email": "batman@example.com"},
]

assert find_contact(contacts, "name", "Дмитрий Менделеев") == []
assert find_contact(contacts, "name", "Пушкин") == []
assert find_contact(contacts, "phone", "910") == []

assert find_contact(contacts, "name", "александр пушкин") == [
    {"name": "Александр Пушкин", "phone": "+7 910 56-78", "email": "pushkin@example.com"},
]
assert find_contact(contacts, "email", "batman@example.com") == [
    {"name": "Брюс Уэйн", "phone": "+7 987 654", "email": "batman@example.com"},
    {"name": "Бэтмен", "phone": "+7 gotham", "email": "batman@example.com"},
]
```
