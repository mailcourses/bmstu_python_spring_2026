# Домашнее задание #01

### 1. Функция для решения квадратного уравнения

Функция должна принимать коэффициенты -a, b, c- квадратного уравнения
```
a*x^2 + b*x + c = 0
```
и возвращать кортеж квадратных корней или None, если корней нет.


### 2. Функция для разделения четных и нечетных чисел

Функция должна принимать на вход список чисел,
разделять его на два списка: четных и нечетных чисел,
и возвращать кортеж из этих двух списков.


### 3. Функция для подсчета слов в тексте

Функция должна принимать текст в виде строки и число k, считать частоту каждого слова в тексте без учета регистра и возвращать k наиболее частотных слов и их частоты.
Использовать стандартную библиотеку для данной задачи нельзя.
